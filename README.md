## üé® Customization

### Using This Template

You're welcome to use this portfolio as a template for your own website! Here's how:

1. **Fork or Clone** this repository
2. **Replace Personal Information:**
   - Update name, email, and contact details
   - Replace profile picture in `images/` folder
   - Update social media links (GitHub, LinkedIn, etc.)
   - Modify projects, skills, and experience
   - Update meta tags in `index.html`
3. **Update URLs:**
   - Change GitHub Pages URL in `sitemap.xml`
   - Update canonical URLs in `index.html`
   - Modify `robots.txt` with your domain
4. **Customize Design** (optional):
   - Modify colors in CSS variables (`:root` section)
   - Adjust fonts, spacing, or layout
   - Add or remove sections as needed

### Important Files to Update

- `index.html` - All personal content and meta tags
- `images/profile_pic.jpeg` - Your profile picture
- `sitemap.xml` - Your website URL
- `robots.txt` - Your website URL
- `.well-known/security.txt` - Your contact email
